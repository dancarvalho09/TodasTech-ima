
> API: Application Programming Interface

  - Conceito: é um sistema que pode realizar diversas requisições

  - Requisições:
    POST:      Cadastrar
    GET:       Selecionar
    PUT:       Alterar
    DELETE:    Remover


  - Padrão de dados (envio e recebimento): JSON
